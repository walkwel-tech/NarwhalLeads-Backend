# narwhal-leads
This is the base project for node with typescript.

## Technologies used
- typescript
- node/express
- mongodb
- mongoose


## Getting started

- Installation

```
git clone http://walkwel-gitlab.local/walkwel-javascript/node-base.git
cd node-base
npm install

```

- Development

```
npm run dev

```
- Build

```
npm run build

```

- Documentation

Import nodeBase.postman_collection.json to postman

  - category-data
  - invoice-pdf
  - logo
